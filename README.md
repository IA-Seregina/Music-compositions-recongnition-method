# Music-compositions-recongnition-method
Ноутбук оформлен как воспроизводимый сценарий улучшенного метода распознавания музыкальных произведений на основе спектральных пиков и хэширования.

**Что делает ноутбук**
- выполняет предобработку и нормализацию аудио;
- строит STFT-спектрограммы и базовые аудио-признаки;
- извлекает спектральные пики и формирует аудио-фингерпринты;
- индексирует базу треков и распознаёт запросы;
- оценивает качество метода по accuracy, времени поиска и устойчивости к искажениям.

**Как запускать**
- выполнять ячейки сверху вниз;
- убедиться, что каталоги `tracks` и `query` доступны из директории ноутбука.
